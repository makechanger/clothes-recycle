# 衣物回收平台需求文档
## （一）用户微信小程序端
### 1. 登录与个人信息管理模块
- 微信授权一键登录，手机号绑定（必填）；
- 个人信息管理：姓名、所属社区（必填项）、常用上门地址；
- 账号注销功能：账号注销前需结清所有未完成订单、无未处理完的申诉 / 投诉，注注销后对个人敏感信息进行脱敏处理，并在法定存储期满后予以彻底删除。
### 2. 核心上门回收预约模块
- 预约信息填写：支持选择上门地址、上门时间段（精确到 2 小时区间）、预估衣物重量、衣物品类，上传衣物照片，补充特殊备注；
- 智能预检功能：通过拍照图像识别，自动识别衣物材质、新旧程度，提前告知用户是否符合回收标准、预估重量，仅做预筛选与参考，最终合规结果以上门回收员现场核验为准，避免无效预约；补充对用户的提示：「预估重量与实际重量偏差超过 ±30% 时，最终结果以现场核验为准」；
- 回收规则清晰公示：明确可回收衣物范围，预估重量仅做参考，以现场实际核验的合格衣物重量为准；同步公示积分获取规则（如：每公斤合格衣物可获得对应积分）；
- 机构选择功能：展示平台审核通过的第三方回收机构（移动端）列表，包含机构名称、机构简介、衣物接收范围、过往服务评价，用户可根据需求选择意向机构，作为衣物的最终去向；
- 订单管理功能：支持订单取消、联系回收员，用户拥有自主取消权，每日取消订单上限为 3 次，超出后当日无法发起新的取消操作，避免恶意操作；回收员超出预约时间段 30 分钟未上门，用户可直接取消订单，无需计入当日取消次数限制；
- 上门服务保障：接单后向用户展示回收员实名认证信息，保障用户人身与财产安全。
### 3. 透明化回收与溯源模块
- 称重数据实时同步：称重数据经用户与回收员双方现场确认后上传至订单，锁定后不可篡改；同时要求回收员上传称重照片留证，用户可实时查看；
- 一物一码溯源核心功能：称重与衣物信息经用户与回收员双方确认后，系统生成该笔订单唯一溯源二维码，与订单号、用户信息、回收员信息、称重数据、衣物品类、对应第三方机构信息强关联，锁定后一般情况下不可篡改；
- 实时展示订单接单、上门确认、回收完成、回收员分发、机构回收完成全环节状态，同步展示每个环节的操作人、操作时间、操作地点记录，用户可通过订单详情查看二维码全链路流转记录；
- 订单在第三方机构确认接收完成后，系统根据最终合格衣物重量发放积分，积分发放默认按“回收员录入重量”执行，若机构确认重量与回收员录入重量不一致，系统以机构确认重量重新核算积分，积分实时到账，用户可在订单详情、个人中心查看积分获取明细。
### 4. 激励与公益体系模块
- 绿色环保账户：累计个人回收总量，生成月度 / 年度环保报告，展示社区环保排行榜；
- 积分管理系统：明确积分获取规则，展示积分余额、积分收支明细（获取、兑换、扣除），支持积分查询；
- 积分商城模块：支持积分兑换商品，商品分类展示，明确每件商品的积分兑换门槛、库存数量、兑换规则；支持商品搜索、收藏、加入兑换车；
- 兑换流程：用户选择商品→确认兑换（扣除对应积分）→填写收货信息→生成兑换订单→用户可跟踪兑换订单物流进度；支持兑换订单取消（未发货状态），取消后自动退还对应积分；积分扣除场景包括：积分兑换、异常订单仲裁后需扣除违规所得积分。
### 5. 服务评价与客服模块
- 服务评价体系：订单完成后，用户可对上门准时度、服务态度、称重规范度进行文字反馈；评价完成后可额外获得少量积分奖励；同时可对选择的第三方机构服务（接收及时性、规范性）进行评价。
- 异常订单处理：支持称重不符、服务不规范、衣物丢失、积分发放异常等问题的申诉，平台介入仲裁并跟踪处理进度，明确申诉成立后的处理规则：根据仲裁结果，支持订单金额、积分的补发 / 扣除，同步更新订单状态，并将最终处理结果反馈给用户。
### 6. 个人中心模块
- 我的回收：全部回收订单记录、详情查询、进度跟踪、对应溯源二维码查看、全链路流转记录调取；
- 我的积分：积分余额查询、积分收支明细、积分商城入口；
- 积分兑换订单：查看所有积分兑换订单记录、兑换详情、物流进度、取消兑换操作；
- 地址管理：常用上门地址、预约地址、积分兑换收货地址的新增、编辑、删除；
- 我的评价：历史服务评价记录。
- 机构查询：可查看平台所有审核通过的第三方回收机构详情、自身订单对应机构的回收完成状态。
## （二）回收员端（小程序端）
### 1. 账号与权限管理
- 专属账号登录：支持手机号验证码登录；
- 资质管理：回收员实名认证平台审核通过后方可接单；
- 个人信息管理：个人联系方式、服务区域、服务时间段设置与修改。
### 2. 订单智能管理模块
- 接单操作：支持一键接单，接单后可查看用户完整上门地址、联系方式、预约信息及用户选择（或平台分配）的第三方机构信息，无合理理由不得拒绝接单；
- 扫码核验功能：支持扫描订单绑定的溯源二维码，快速调取订单详情、衣物信息、对应第三方机构信息，完成交接、分发环节的信息核验与状态更新，扫码记录实时同步至系统、用户端及对应第三方机构端；
- 异常订单上报：回收员在触发现场异常（如用户失联、拒收）时，可申请‘异常结单’，经平台审核后生效
- 分发任务管理：接收平台分配的衣物分发任务，明确分发目的地（用户选择的第三方机构地址），可查看机构联系方式、接收时间要求，完成分发后需扫码确认，同步更新订单状态为 “已分发”。
### 3. 上门服务执行模块
- 称重录入：支持手动录入，需同步上传带清晰刻度的称重照片留证，录入数据需经用户现场确认，确认后锁定不可篡改；称重数据直接关联用户积分核算，录入错误需上报平台审核后调整；
- 打包赋码功能：称重确认后，系统自动生成唯一溯源二维码，回收员仅打印粘贴，将其与当前已确认的订单信息进行在线绑定，并粘贴至封口处，绑定后一般情况下不可修改、不可解绑；
- 订单完成确认：回收完成、赋码绑定成功后，提交订单完成申请，用户确认仅作为服务确认节点，不作为积分发放依据，订单在第三方机构确认接收完成后闭环，系统统一触发积分发放。
- 物料交接与分发管理：回收员需按订单指定的第三方机构，完成衣物分发，与机构交接时，必须逐单扫描溯源二维码完成核验，确认衣物包裹完好、二维码无拆改，扫码后更新订单交接状态为 “机构已接收”，记录交接人、交接时间、交接地点，生成电子交接单，平台全程留痕，无扫码交接记录的订单无法完成回收闭环。
## （三）第三方机构端（小程序端）
### 1. 账号与权限管理
- 专属账号登录：支持手机号验证码登录，账号由平台审核通过后开通；
- 资质管理：入驻时提交营业执照、回收资质、场地证明等材料，平台审核通过后方可开展接收业务；资质到期前系统自动提醒，需及时提交更新材料；
- 个人 / 机构信息管理：维护机构基础信息（名称、简介、地址、联系方式、接收范围、服务时间），可修改联系方式、接收时间等非核心信息，核心信息（名称、资质）修改需提交平台审核。
### 2. 申请与入驻模块
- 入驻申请：填写机构基础信息、上传相关资质证明，提交平台审核，可在移动端查看审核进度、审核意见；
- 资质更新：资质到期前，上传更新后的资质材料，提交平台审核，确保机构合规运营；可查看资质更新审核进度、审核意见。
- 账号状态查看：查看自身账号状态（正常、审核中、禁用），接收平台下发的违规提醒、整改通知。
### 3. 衣物接收与管理模块
- 接收任务查看：查看平台分配及用户选择的衣物接收任务，包含订单详情、衣物信息、溯源二维码、回收员联系方式、预计分发时间；
- 扫码接收：回收员送达后，扫描衣物溯源二维码完成核验，确认衣物包裹完好、信息一致后，点击 “确认接收”，同步更新订单状态为 “机构已接收”，生成接收记录，同步至系统、用户端、回收员端；
- 异常接收上报：若衣物存在破损、数量不符、二维码异常等问题，可上传凭证，提交异常接收申请，反馈给平台及回收员，等待平台仲裁；
- 衣物入库管理：接收衣物并确认无误后，在移动端反馈衣物回收完成状态，标注回收完成时间，同步至系统及用户端，供用户查看溯源。
### 4. 数据查询与统计模块
- 接收数据查询：查看自身接收的衣物总量、订单数、每日 / 每月接收明细，可按时间、衣物品类多维度筛选；
- 用户评价查看：查看用户对本机构的服务评价，可针对负面评价进行回复、整改，整改情况可提交平台备案；
- 异常记录查询：查看自身上报的异常接收记录、平台下发的违规记录，了解处理结果及整改要求。
## （四）平台系统管理员 Web 管理后台
### 1. 全域权限与账号管理
- 账号管理：回收员账号、第三方机构账号的新增、编辑、密码重置；
- 溯源权限配置：为回收员、第三方机构配置对应扫码操作权限，无权限修改二维码绑定信息；
- 积分权限配置：为运营人员配置积分规则修改、积分发放 / 扣除审核、积分商城商品管理等权限；
- 机构权限配置：为第三方机构配置移动端操作权限（申请提交、订单查看、衣物接收、状态更新等）。
### 2. 回收员全生命周期管理
- 入驻管理：回收员入驻申请、实名认证管理；
- 服务管控：回收员投诉处理、溯源赋码违规操作记录管理、积分核算相关违规处理。
### 3. 订单全流程管理
- 订单管理：全平台所有回收订单的查看、进度跟踪、历史记录导出，支持按区域、时间、状态、对应第三方机构多维度筛选，可查看每笔订单对应的溯源二维码信息、全链路扫码记录、积分发放明细、分发及接收记录；
- 异常订单处理：用户申诉、回收员上报的异常订单的平台仲裁、处理跟进，处理结果同步给用户、回收员及对应第三方机构；异常订单人工调整：针对异常订单（用户申诉、回收员上报），支持人工审核与调整，涉及积分异常的，可手动补发、扣除积分，并留存操作记录；
### 4. 基础信息与合作方管理
- 社区信息管理：支持社区范围、名称的新增、编辑、删除，维护全平台社区基础信息，为用户社区绑定、环保排行榜提供数据支撑；
- 内容管理：科普内容、公益项目、平台公告的审核、发布、上下线全流程管理；同步支持积分规则、积分商城规则的编辑、发布、更新；
- 第三方机构管理：负责第三方机构入驻申请审核、基础信息维护及账号状态管控，赋予机构移动端核心操作权限，全流程可追溯；
- 积分体系管理：配置积分获取规则（如：不同品类、重量的积分系数）、积分扣除规则，支持规则的新增、编辑、生效 / 失效操作；审核积分补发、扣除申请，留存审核记录；
- 积分商城管理：支持积分兑换商品的新增、编辑、上下线、库存管理，设置商品兑换积分门槛、兑换规则、物流配置；查看积分兑换订单，处理兑换订单取消、退款（积分退还）、物流异常等问题；统计商品兑换数据。
### 5. 数据统计与分析
- 核心经营数据：全平台回收总量、日 / 周 / 月 / 年订单量、完单率、客单价、区域回收排行、衣物去向分布（按第三方机构分类统计）、机构选择偏好；
- 用户数据：新增用户数、活跃用户数、留存率、预约频次、用户画像分析；
- 回收员数据：全平台回收员、接单量、异常订单率；
- 第三方机构数据：机构数量、衣物接收量、用户评价评分；
- 公益数据：捐赠衣物总量；
- 积分相关数据：积分总发放量、积分总兑换量、积分兑换率、热门兑换商品排行；
- 可视化报表：支持多维度数据筛选、自定义报表导出、运营看板配置，包含积分相关数据看板，便于运营人员实时监控。
